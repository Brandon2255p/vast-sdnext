# VAST

This project is a Docker container for running SDNext (Stable Diffusion Next) on
Vast.ai.

## Prerequisites

- A Vast.ai account
- A GPU-enabled instance on Vast.ai

## Usage

1. Deploy this Dockerfile on a Vast.ai GPU instance
2. The container will automatically clone SDNext from
   `https://github.com/vladmandic/sdnext`
3. Access the Stable Diffusion web interface through the appropriate port
